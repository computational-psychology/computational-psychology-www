HOWTO
------


Requirements: 
pelican and ghp-import
They can be installed with: 'pip install pelican ghp-import' 



To clone the repository:

git clone --recursive  git@github.com:computational-psychology/computational-psychology.github.io.git




To edit:

Run 'make devserver'  in the repository root folder. 
It compiles the website and updates it when a change has been detected.

The website can be seen at http://localhost:8000

Content reside in content/ folder, pages are in RST format.




To publish:

- Add, commit and push your changes as you would normally do in git
(git commit -m "message") .. and push it (git push)

- run ./publish.sh  to deploy the latest website version online
